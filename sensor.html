Sensor

Sensors are how ships see the world around them.  There are different sensors are for different functions.

Examples of sensors

  • Combat sensor - Fast IFF providing general ship ship information
  • Mining sensor - Slower detailed for detecting different types of mineral compositions
  • Bounty Hunting sensor - Provides fast ship + cargo ID
  • Cloak Detection sensor - slow sensor for finding cloaked ships which do not wish to be found.
